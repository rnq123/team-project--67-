# Документация функций

## Модуль game.py

### Класс GuessNumberGame

#### `__init__(self)`
Инициализирует объект игры.

**Параметры:** Нет
**Возвращает:** None
**Описание:** Создает экземпляр RecordsManager и устанавливает начальные значения атрибутов.

#### `show_menu(self)`
Отображает главное меню игры.

**Параметры:** Нет
**Возвращает:** None
**Выводит:** Главное меню в консоль

#### `get_player_name(self)`
Запрашивает имя игрока.

**Параметры:** Нет
**Возвращает:** str - имя игрока
**Исключения:** Запрашивает повторный ввод при пустом имени

#### `select_difficulty(self)`
Позволяет выбрать уровень сложности.

**Параметры:** Нет
**Возвращает:** tuple - (уровень, параметры уровня)
**Исключения:** ValueError при некорректном вводе

#### `setup_game(self, level_params)`
Настраивает игру согласно выбранному уровню.

**Параметры:**
- `level_params` (dict): Параметры уровня сложности

**Возвращает:** None
**Описание:** Устанавливает диапазон чисел, количество попыток и загадывает число.

#### `play_round(self)`
Выполняет один игровой раунд.

**Параметры:** Нет
**Возвращает:** bool - True если игрок выиграл, False если проиграл
**Исключения:** ValueError при некорректном вводе числа

#### `show_instructions(self)`
Показывает инструкцию к игре.

**Параметры:** Нет
**Возвращает:** None
**Выводит:** Инструкцию в консоль

#### `run(self)`
Основной игровой цикл.

**Параметры:** Нет
**Возвращает:** None
**Описание:** Управляет всем игровым процессом.

## Модуль records.py

### Класс RecordsManager

#### `__init__(self, filename="records.json")`
Инициализирует менеджер рекордов.

**Параметры:**
- `filename` (str): Имя файла для сохранения рекордов

**Возвращает:** None

#### `load_records(self)`
Загружает рекорды из файла.

**Параметры:** Нет
**Возвращает:** dict - словарь с рекордами
**Исключения:** Возвращает пустой словарь при ошибке чтения

#### `save_records(self)`
Сохраняет рекорды в файл.

**Параметры:** Нет
**Возвращает:** bool - True если успешно, False при ошибке
**Исключения:** Выводит сообщение об ошибке

#### `add_record(self, player_name, difficulty, attempts, max_number)`
Добавляет новый рекорд.

**Параметры:**
- `player_name` (str): Имя игрока
- `difficulty` (str): Уровень сложности
- `attempts` (int): Количество попыток
- `max_number` (int): Максимальное число в диапазоне

**Возвращает:** None
**Описание:** Добавляет запись, сортирует и сохраняет.

#### `display_records(self)`
Отображает таблицу рекордов.

**Параметры:** Нет
**Возвращает:** None
**Выводит:** Таблицу рекордов в консоль

## Вспомогательные функции

### `main()`
Точка входа в программу.

**Параметры:** Нет
**Возвращает:** None
**Описание:** Создает экземпляр игры и запускает его.

## Конфигурационные константы

### В config.py:

#### `DIFFICULTY_LEVELS`
Словарь с параметрами уровней сложности:
- ключ: название уровня (str)
- значение: словарь с параметрами:
  - `range` (int): максимальное число
  - `attempts` (int): количество попыток
  - `description` (str): описание уровня

#### `RECORDS_FILE`
Имя файла для сохранения рекордов (по умолчанию: "records.json")
